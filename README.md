## Hi there 👋

I'm Geoffrey, a frontend developer based in France.

I've been passionate about the web since I was a kid, studied web development early on, then took a detour through digital marketing. That detour taught me a lot, like how users think, what makes a product actually useful and how to communicate clearly.

But I always wanted to go back to building things, so I did. I just completed my JavaScript & React Developer one year online course.

Over the past months, I worked on real projects, including:

- [Booki](https://github.com/geoffreybunel/booki) A travel agency homepage (HTML & CSS)
- [GameOn](https://github.com/geoffreybunel/GameOn-website-FR) A landing page with JavaScript
- [FisheEye](https://github.com/geoffreybunel/Front-End-Fisheye) An accessible website for a photographers' platform
- [Les Petits Plats](https://github.com/geoffreybunel/les-petits-plats) A search algorithm in JavaScript for a recipe website
- [Bill](https://github.com/geoffreybunel/Billed-app-FR-Front) Debugging & testing an HR SaaS
- [Kasa](https://github.com/geoffreybunel/kasa) A full web app with React & React Router
- [SportSee](https://github.com/geoffreybunel/sportsee) An analytics dashboard with React
- [ArgentBank](https://github.com/geoffreybunel/Argent-Bank) A bank account interface consuming a REST API
- [HRNet](https://github.com/geoffreybunel/HRNet) Migrating a jQuery library to React
